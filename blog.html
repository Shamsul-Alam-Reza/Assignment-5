Question-1:What are the different ways to select an element in the DOM?

getElementById()
getElementsByClassName()
getElementsByTagName()
querySelector()
querySelectorAll()

Question-3:What is event delegation in the DOM?

Delegation is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event triggered on a child element propagates upwards through its parent elements, eventually reaching the root of the document

Question-5:How do you create, add, and remove elements using JavaScript?

Create an element: const element = document.createElement('div');
Add an element to the DOM: document.body.appendChild(element);
Remove an element from the DOM: element.remove();
